# Security Policy

## Supported Branches

The repository is currently maintained on these branches:

| Branch | Status |
| --- | --- |
| `production` | Supported |
| `main` | Supported |

Feature branches and older commits are not considered supported release lines.

## Reporting a Vulnerability

Do not open public GitHub issues for security reports.

If GitHub Private Vulnerability Reporting is enabled for this repository, use that flow first. Otherwise, contact the maintainer privately through GitHub and include:

- a short description of the issue
- impact and affected area
- reproduction steps or proof of concept
- any suggested mitigation if you have one

Please avoid publishing exploit details until the issue has been triaged and fixed.
